(static) construct

Constructs an array for a given value and CSS property. The array is made up of strings and themeGet calls. This function is an abstraction to be used for functions representing CSS properties.
Default Value:
  • px

(static) px

Transforms numbers into px values and leaves strings as they are. This is the default transformer for the construct function.
Source:
  • lib/util/transformers/px.js, line 1
Example
const pxValue = px(100)
const stringValue = px('1rem')

pxValue === '100px'
stringValue === '1rem'
